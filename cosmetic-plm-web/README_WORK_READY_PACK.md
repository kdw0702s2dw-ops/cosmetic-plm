# Work Ready Integrated Pack

고객 포털은 제외하고, 출근 직후 바로 사용할 수 있는 5가지 기능을 통합했습니다.

Included:
1. 실제 데이터 연동 준비
2. Cosmetic AI Brain
3. 문서 자동 생성
4. PLM Chatbot
5. 코드 품질 / 배포 안정성 점검

Apply:
1. 압축 해제
2. cosmetic-plm-web 폴더에 덮어쓰기
3. Supabase SQL Editor에서 supabase/migrations/work_ready_integrated_pack.sql 실행
4. npm run build
5. git add .
6. git commit -m "add work ready integrated pack"
7. git push

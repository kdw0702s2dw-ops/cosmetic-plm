# Master Data Live CRUD Pack

원료마스터, INCI, 처방마스터, 규제룰을 실제 업무용 CRUD 화면으로 관리하기 위한 패키지입니다.

Included:
1. Raw Material Live CRUD
2. INCI Master Live CRUD
3. Formula Master Live CRUD
4. Regulation Rules Live CRUD
5. Search / Duplicate Check / Normalize Formula 100%
6. CRUD Audit Trail preparation

Apply:
1. 압축 해제
2. cosmetic-plm-web 폴더에 덮어쓰기
3. Supabase SQL Editor에서 supabase/migrations/master_data_live_crud_pack.sql 실행
4. npm run build
5. git add .
6. git commit -m "add master data live crud pack"
7. git push

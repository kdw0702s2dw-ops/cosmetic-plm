# Enterprise Go-Live Final Pack

출근 전 최종 점검용 패키지입니다.

추가 화면:
- /enterprise-go-live

적용 방법:
1. 압축 해제
2. cosmetic-plm-web 폴더에 그대로 덮어쓰기
3. Supabase SQL Editor에서 아래 파일 실행
   supabase/migrations/enterprise_go_live_final_pack.sql
4. npm run build
5. git add .
6. git commit -m "add enterprise go live final pack"
7. git push

접속:
- http://localhost:3000/enterprise-go-live
- https://도메인/enterprise-go-live

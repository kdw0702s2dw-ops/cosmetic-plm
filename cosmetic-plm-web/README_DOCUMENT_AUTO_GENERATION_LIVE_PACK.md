# Document Auto Generation Live Pack

Formula Live Calculation 결과를 바탕으로 업무 문서를 자동 생성합니다.

Included:
1. Formula Sheet
2. Ingredient Composition
3. Full Ingredient List
4. Product Specification Draft
5. Test Request Draft
6. Development Report Draft
7. Customer Summary
8. Document Approval
9. Download-ready status
10. Document risk check

Apply:
1. 압축 해제
2. cosmetic-plm-web 폴더에 덮어쓰기
3. Supabase SQL Editor에서 supabase/migrations/document_auto_generation_live_pack.sql 실행
4. npm run build
5. git add .
6. git commit -m "add document auto generation live pack"
7. git push

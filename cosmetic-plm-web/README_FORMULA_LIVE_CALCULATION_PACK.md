# Formula Live Calculation Pack

처방 작성 시 필요한 계산 기능을 통합했습니다.

Included:
1. Formula total percentage validation
2. Cost per kg calculation
3. Batch size raw material requirement
4. Complex ingredient breakdown
5. Full ingredient list generation
6. Regulation risk display
7. Pre-save validation checklist

Apply:
1. 압축 해제
2. cosmetic-plm-web 폴더에 덮어쓰기
3. Supabase SQL Editor에서 supabase/migrations/formula_live_calculation_pack.sql 실행
4. npm run build
5. git add .
6. git commit -m "add formula live calculation pack"
7. git push

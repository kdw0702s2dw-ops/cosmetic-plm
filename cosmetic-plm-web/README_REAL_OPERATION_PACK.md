# Real Operation Pack

실제 ODM 연구소 운영에 필요한 기능을 통합했습니다.

Included:
1. Quick Access
2. Excel 대량등록 검증
3. 빠른 통합 검색
4. 즐겨찾기 / 최근 작업
5. 오늘 할 일 / 업무 알림
6. 성능 최적화 / 운영 안정성

Apply:
1. 압축 해제
2. cosmetic-plm-web 폴더에 덮어쓰기
3. Supabase SQL Editor에서 supabase/migrations/real_operation_pack.sql 실행
4. npm run build
5. git add .
6. git commit -m "add real operation pack"
7. git push

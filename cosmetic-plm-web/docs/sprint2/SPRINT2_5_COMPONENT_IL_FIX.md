# Sprint 2-5 복합성분표 / 전성분표 수정

수정 내용:
- 기존 복합성분표가 INCI 문자열의 쉼표 여부로 복합성분을 판단하던 문제 수정
- 이제 `plm_raw_material_components`를 직접 조회해서 복합원료 구성성분을 전개
- 복합성분표 컬럼:
  - 국문 INCI
  - 영문 INCI
  - CAS No.
  - EC No.
  - 기능
  - 함량(%)
- 원료코드, 원료명은 복합성분표 출력에서 제외
- 함량은 `처방 투입량 × 원료 내 구성비 ÷ 100`으로 계산
- 동일 성분은 합산 표시
- 전성분표도 복합원료 구성성분 + 단일성분을 합산해 국문/영문 INCI List 출력

주의:
- 복합성분표가 비어 있으면 원료관리에서 해당 원료의 구성성분을 먼저 등록해야 합니다.
